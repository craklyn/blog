StarMap

A project that draws the stars in the sky and allows users to define cuts on which stars are drawn.

I imported astronomy database HYG3.0 into a local MySQL database. I wrote an app in the Processing Language to query the database and draw output to customized interface.

RavenLOVE

A prototype game I coded. My friend, Nathan Aldrich, drew the bitmap art.

The game was made in the Love2D game engine and made use of the Advanced Tile Loader library. We created basic levels, enemies, and proof of concept. Game coded in LUA.
